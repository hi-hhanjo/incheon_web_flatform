# 컴포넌트: SongCard (응원가 카드)

- **문서 버전**: v1.3 · **작성일**: 2026-07-11 (v1.2 갱신: 2026-07-17)
- **연관 문서**: BASE.md, FUNCTION.md (F-01, v1.3 정렬·배지), badge.md
- **한 줄 요약**: 목록에서 응원가 한 곡을 나타내는 카드. 제목과 배지(미사용·대표곡)를 보여주고, 누르면 상세로 이동합니다.

---

## 1. 개요
- 목록 화면에서 **곡 하나 = 카드 하나**입니다.
- 곡 제목을 보여주고, 대표곡이면 배지를 함께 표시합니다. 카드를 누르면 해당 곡 상세 화면으로 넘어갑니다.

## 2. 사용처
- F-01 목록 화면 (곡 개수만큼 반복 생성)

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| id | 곡 고유 번호 (상세 이동 시 전달) | O |
| title | 곡 제목 | O |
| tags | 태그 목록 (예: ["미사용"], ["대표곡"]) | X |
| onClick | 카드를 눌렀을 때 상세로 이동 | O |

## 4. 구성 요소
- 곡 **제목** (필수)
- **미사용 배지** (tags에 '미사용'이 있을 때만 표시 → `badge.md` 사용) *(v1.1 신설)* — 팀을 떠난 선수의 응원가라 지금은 경기장에서 불리지 않는다는 표시입니다. 이 곡들은 F-01 목록에서 항상 맨 아래 묶음에 놓입니다(FUNCTION.md v1.3). 판정은 구단 공식 선수단 명단 크롤로 자동화돼 있습니다(`scraper/classify_chants.py`).
- **대표곡 배지** (tags에 '대표곡'이 있을 때만 표시 → `badge.md` 사용)
- **스트라이프 악센트**: 카드 왼쪽에 붙는 세로 바. 목록에서 카드 순서(0부터)가 짝수면 인천 블루(`#2E7DFF`), 홀수면 보더 라이트(`#414A5A`, '검정' 대표색)로 번갈아 표시 (BASE.md "스트라이프 악센트" 참고)

## 5. 상태 (States)
| 상태 | 처리 |
|------|------|
| 기본 | `#161A21` 배경 카드 |
| 호버/누름 | 배경이 살짝 밝아짐 (`#1E232C`) |

## 6. 스타일 규칙 (BASE.md 기준)
- 배경: `#161A21`, 모서리: 12px, 테두리 없음
- 제목: 16px / 600 / `#FFFFFF` / Pretendard
- 안쪽 여백: 16px
- 카드 사이 세로 간격: 12px
- **최소 높이 44px 이상** 확보 (터치 편의, F-03)
- 스트라이프 악센트: 두께 8px, 완전히 둥근(pill) 세로 바, 카드 본문과 12px 간격. 색은 본문 배경·모서리에 영향을 주지 않고 별도 요소로만 표시

## 7. 동작
1. 사용자가 카드를 탭한다.
2. 해당 곡의 `id`를 들고 상세 화면(F-02)으로 이동한다.

## 8. 사용 예시 (의사코드)
```
songs.map(song =>
  <SongCard
    id={song.id}
    title={song.title}
    tags={song.tags}
    onClick={() => 상세로이동(song.id)}
  />
)
```
