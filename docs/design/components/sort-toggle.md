# 컴포넌트: SortToggle (정렬 전환 토글)

- **문서 버전**: v1.0 · **작성일**: 2026-07-13
- **연관 문서**: DESIGN.md, 기능명세서.md (F-01, v1.1)
- **한 줄 요약**: 응원가 목록을 '기본'과 '가나다순' 중 하나로 전환하는 세그먼트 토글 버튼.

---

## 1. 개요
- 목록 화면(F-01)의 기본 정렬은 등록 순서입니다. 이 컴포넌트는 사용자가 필요할 때 제목 기준 가나다순으로 바꿔 볼 수 있게 합니다.
- 시각적 구조는 `video-tabs.md`(영상 선택 세그먼트 탭)와 동일한 패턴을 재사용합니다.

## 2. 사용처
- F-01 목록 화면, 상단 제목과 카드 목록 사이 (곡이 1개 이상일 때만 표시)

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| sortMode | 현재 정렬 모드 (`"default"` \| `"alphabetical"`) | O |
| onChange | 옵션을 눌렀을 때 정렬 모드 변경 | O |

## 4. 동작
1. 기본값은 `"default"`(기본)이다.
2. '가나다순'을 누르면 `onChange("alphabetical")`가 호출되어 목록이 제목 기준으로 재정렬된다.
3. '기본'을 다시 누르면 원래 순서로 돌아온다.
4. 곡이 0개(빈 상태)인 경우 이 컴포넌트 자체를 렌더링하지 않는다.

## 5. 스타일 규칙 (DESIGN.md 기준, video-tabs.md와 동일 패턴)
- 컨테이너: `#1E232C` 배경, 9999px(완전 둥근 pill)
- 선택된 옵션: 인천 블루(`#2E7DFF`) 배경 + 화이트 텍스트
- 선택 안 된 옵션: 투명 배경 + 실버(`#A8B0BD`) 텍스트
- 글자: 15px / 600 / Pretendard
- 버튼 높이: 최소 44px (F-03)

## 6. 접근성
- 각 옵션 버튼에 `aria-pressed`로 현재 선택 상태를 표시한다.

## 7. 사용 예시 (의사코드)
```
<SortToggle sortMode={정렬모드} onChange={(mode) => 정렬모드변경(mode)} />
```
