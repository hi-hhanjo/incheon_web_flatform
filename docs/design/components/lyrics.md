# 컴포넌트: Lyrics (가사 영역)

- **문서 버전**: v1.0 · **작성일**: 2026-07-11
- **연관 문서**: DESIGN.md (3·4장), 기능명세서.md (F-02)
- **한 줄 요약**: 응원가 가사를 줄바꿈 그대로, 따라 부르기 편한 크기·행간으로 보여줍니다.

---

## 1. 개요
- 상세 화면에서 영상 아래에 오는 **가사 표시 영역**입니다.
- 이 서비스의 가사는 '읽고 따라 부르는' 콘텐츠라, **글자를 크게·행간을 넉넉하게** 잡는 것이 핵심입니다.

## 2. 사용처
- F-02 상세 화면 (영상 아래)

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| text | 가사 내용 (줄바꿈 `\n` 포함) | O |

## 4. 스타일 규칙 (DESIGN.md 기준)
- 배경: `#0E1116` (베이스와 동일 → 텍스트에 집중)
- 글자: **17px / 400 / 행간 1.75** / `#FFFFFF` / Pretendard
- **가운데 정렬** (text-align: center)
- **줄바꿈(`\n`)을 원본 그대로 표시** (문단 유지)
- 위아래 여백: 넉넉하게 (예: 20~24px)
- 왼쪽에 보더 라이트(`#414A5A`, '검정' 대표색) 스트라이프 악센트 고정 배치 (두께 8px, pill, 12px 간격 — DESIGN.md "스트라이프 악센트" 참고)

> **핵심**: 다른 텍스트보다 크고(17px) 행간이 넓습니다(1.75). Spotify식 촘촘한 행간을 쓰지 않습니다. (DESIGN.md의 의도적 차이)

## 5. 상태별 처리
| 상황 | 처리 |
|------|------|
| 정상 | 가사를 줄바꿈대로 표시 |
| 가사 없음(빈 값) | `feedback.md`의 "가사 준비 중입니다" 표시 |

## 6. 사용 예시 (의사코드)
```
{song.lyrics
  ? <Lyrics text={song.lyrics} />
  : <Feedback type="empty" message="가사 준비 중입니다" />
}
```
