# 컴포넌트: YoutubePlayer (유튜브 영상)

- **문서 버전**: v1.0 · **작성일**: 2026-07-11
- **연관 문서**: DESIGN.md, TECHSTACK.md (2.4), 기능명세서.md (F-02, F-04)
- **한 줄 요약**: 유튜브 영상 ID를 받아 임베드(iframe)로 재생하는 영역. 16:9 비율을 유지합니다.

---

## 1. 개요
- 유튜브에 있는 원본 영상을 **창문처럼 비춰 보여주는** 영역입니다. (영상을 복사해오지 않음 → 저작권 안전)
- `youtubeId` 하나만 있으면 재생됩니다.

## 2. 사용처
- F-02 상세 화면 (VideoTabs에서 선택된 영상을 표시)

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| youtubeId | 유튜브 영상 ID (예: dQw4w9WgXcQ) | O |

## 4. 스타일 규칙 (DESIGN.md 기준)
- **가로세로 비율 16:9 유지** — 화면 너비에 맞춰 크기 자동 조절 (F-03)
- 모서리: 12px
- 로딩 중 배경: `#161A21`

## 5. 상태별 처리
| 상황 | 처리 |
|------|------|
| 정상 | 영상 임베드 표시 |
| youtubeId 없음 | 이 컴포넌트를 그리지 않고, 상위에서 `feedback.md`의 "영상 준비 중입니다" 표시 |

## 6. 구현 메모 (초보자용)
- 유튜브 임베드 주소 형태: `https://www.youtube.com/embed/{youtubeId}`
- iframe에 `allowfullscreen`을 켜서 전체화면을 지원합니다.
- 선택된 영상 1개만 불러옵니다. (탭을 바꿀 때만 새 영상 로드 → 성능, PRD 8번)

## 7. 사용 예시 (의사코드)
```
<YoutubePlayer youtubeId={선택된영상.youtubeId} />
```

## 8. AI 지시용 스니펫
> "유튜브 임베드 플레이어: `https://www.youtube.com/embed/{youtubeId}`를 iframe으로. 16:9 비율 유지(폭 100%), 12px 모서리, allowfullscreen. 로딩 배경 `#161A21`. youtubeId 없으면 렌더하지 않음."
