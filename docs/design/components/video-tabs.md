# 컴포넌트: VideoTabs (영상 선택 탭)

- **문서 버전**: v1.1 · **작성일**: 2026-07-11
- **연관 문서**: BASE.md, FUNCTION.md (F-04), PRD.md (10번 의사결정)
- **한 줄 요약**: 공식/현장 등 여러 영상 중 하나를 고르는 세그먼트 탭. `videos` 목록을 읽어 탭을 자동 생성합니다.

---

## 1. 개요
- 한 곡에 영상이 여러 개(공식/현장)일 때, **한 번에 하나만** 골라 보게 하는 탭입니다.
- 탭은 데이터의 `videos` 목록을 읽어 **자동으로** 만들어집니다. (하드코딩하지 않음)

## 2. 사용처
- F-02 상세 화면 상단 (영상 위)

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| videos | 영상 목록 (type, label, youtubeId) | O |
| selectedIndex | 현재 선택된 탭 번호 | O |
| onSelect | 탭을 눌렀을 때 선택 변경 | O |

## 4. 동작 (F-04)
1. 진입 시 **첫 번째 영상(기본: 공식 영상)** 이 선택된 상태로 시작.
2. 다른 탭을 누르면 → 선택을 바꾸고, 해당 영상으로 교체 (youtube-player.md와 연동).
3. 선택된 영상 1개만 불러온다. (나머지는 미로딩)

## 5. 상태별 처리 (중요)
| 상황 | 처리 |
|------|------|
| 영상 2개 이상 | 탭 여러 개 표시 |
| 영상 1개 | **탭을 만들지 않음** (바로 그 영상만 표시) |
| 영상 0개 | 탭 없음 → `feedback.md`의 "영상 준비 중입니다" 표시 |

## 6. 스타일 규칙 (BASE.md 기준)
- 컨테이너: `#1E232C` 배경, 9999px pill
- 선택된 탭: `#2E7DFF` 배경 + `#FFFFFF` 글자
- 선택 안 된 탭: 투명 배경 + `#A8B0BD` 글자
- 글자: 15px / 600 / Pretendard
- 탭 높이: 최소 44px (F-03)

## 7. 사용 예시 (의사코드)
```
<VideoTabs
  videos={song.videos}
  selectedIndex={선택번호}
  onSelect={(i) => 선택번호변경(i)}
/>
```
