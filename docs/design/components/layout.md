# 컴포넌트: Layout (페이지 틀)

- **문서 버전**: v1.0 · **작성일**: 2026-07-11
- **연관 문서**: DESIGN.md, 기능명세서.md (F-03)
- **한 줄 요약**: 모든 화면을 감싸는 모바일 우선 중앙 정렬 틀. 배경색·최대 폭·기본 여백을 책임집니다.

---

## 1. 개요
- 목록 화면·상세 화면 등 **모든 페이지의 가장 바깥을 감싸는** 컨테이너입니다.
- 스마트폰에서는 화면을 꽉 채우고, 태블릿·PC에서는 콘텐츠를 가운데로 모아 너무 넓게 퍼지지 않게 합니다.

## 2. 사용처
- F-01 목록 화면, F-02 상세 화면 (즉, 모든 화면)

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| children | 이 틀 안에 들어갈 화면 내용 | O |

## 4. 스타일 규칙 (DESIGN.md 기준)
- 배경색: `#0E1116` (베이스)
- 최소 높이: 화면 전체(100vh) — 배경이 아래까지 채워지도록
- **최대 콘텐츠 폭**: 약 640px, 화면 가운데 정렬
- 좌우 안쪽 여백: 모바일 16px, 태블릿 이상 20~24px
- 기본 글자색: `#FFFFFF`, 폰트: `Pretendard`

## 5. 반응형 (F-03)
- 모바일(<600px): 폭 100%, 좌우 여백 16px
- 태블릿(600–900px): 콘텐츠 폭 중앙 제한
- PC(>900px): 콘텐츠 폭 최대 640px 중앙 배치
- **어떤 크기에서도 가로 스크롤이 생기지 않아야 함**

## 6. 동작
- 스스로 특별한 동작은 없고, 안에 담긴 화면을 감싸 배경·폭·여백만 잡아줍니다.

## 7. 사용 예시 (의사코드)
```
<Layout>
  <PageTitle>인천 유나이티드 응원가</PageTitle>
  ...화면 내용...
</Layout>
```

## 8. AI 지시용 스니펫
