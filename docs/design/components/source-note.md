# 컴포넌트: SourceNote (출처 표기)

- **문서 버전**: v1.1 · **작성일**: 2026-07-17 (v1.1 갱신: 2026-07-19)
- **연관 문서**: BASE.md (4장 '미세 정보' 역할), PRD.md (9장 저작권·데이터 출처 표기 범위), API_SPEC.md (2장 Source 객체), CRAWLER_SPEC.md (3.7절)
- **한 줄 요약**: "이 데이터는 어디서 가져왔다"를 화면 맨 아래에 가장 작은 글씨로 밝힙니다.

---

## 1. 개요
- **왜 있나**: PRD 9장이 두 가지를 요구합니다 — ① 가사는 출처를 함께 표기할 것(저작권), ② 구단 정보도 어디서 수집하는지 사용자가 알 수 있을 것. 이 부품 하나가 두 요구를 모두 담당합니다.
- **의도적으로 눈에 띄지 않습니다.** 출처는 콘텐츠(가사·순위표)를 읽는 걸 방해하면 안 되는 보조 정보라, BASE.md 4장의 **'미세 정보'(12px / 400 / 행간 1.5)** 역할을 그대로 씁니다. 배지(Badge)와 달리 배경색·테두리가 없습니다.
- 링크가 있으면 밑줄이 붙고, 눌러서 원문으로 갑니다(새 탭).

## 2. 사용처
| 화면 | 라벨 | 출처 |
|------|------|------|
| F-02 상세 (가사 아래) | `가사 출처` | 곡별 `Song.source` — 곡마다 다름 |
| 구단 정보 (`/club`) | `경기 데이터 출처` | 다음 스포츠 — 경기·순위표·상대전적·최근 폼 전부 이 한 줄이 덮음 |

*(v1.1 — 2026-07-17 순위표(구 `/club/standings`)와 다음 상대(구 `/club/opponent`)가 `/club` 한 페이지의 섹션으로 통합됐습니다. 별도 화면·별도 SourceNote였던 `순위표 출처`·`데이터 출처` 두 줄은 사라지고, `/club` 하단의 `경기 데이터 출처` 한 줄(`app/club/page.tsx`)이 모든 구단 데이터를 덮습니다 — 전부 수집처가 다음 스포츠라 6장 규칙대로 한 줄로 합쳤습니다. 각 섹션의 수집 기준일은 SourceNote가 아니라 섹션별 Badge(`snapshotLabel`)가 표시합니다.)*

- **응원가는 곡마다 출처가 다르고**(공식 홈페이지 or 인천네이션), **구단 정보는 데이터셋 단위로 고정**입니다. 그래서 전자는 `Song.source` 필드에서, 후자는 `lib/api/sources.ts`의 상수에서 옵니다.
- `source`가 없는 곡은 **아무것도 표시하지 않습니다** — "출처 없음" 같은 문구를 띄우지 않습니다. *(2026-07-17 기준 34곡 전부 출처가 있어 실제로 이 경우는 없습니다. 손으로 등록했던 id 1·3도 `reconcile_songs.py`가 크롤 결과와 대조해 출처를 채웠습니다. 그래도 `source`는 계속 선택 필드입니다 — 앞으로 손으로 곡을 추가할 수 있기 때문입니다.)*

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| label | 앞에 붙는 설명 (예: "가사 출처") | O |
| name | 출처 이름 (예: "인천네이션") | O |
| url | 원문 주소. 없으면 링크 없이 이름만 표시 | X |

## 4. 표시 형태
```
가사 출처: 인천네이션          ← "인천네이션"에 밑줄, 누르면 원문으로
```

## 5. 상태
- **기본**: 글자색 `#6B7480` (text-muted)
- **마우스 올림**: 링크 글자색이 `#A8B0BD` (text-secondary)로 밝아짐
- 별도의 로딩/에러 상태가 없습니다 — 표시할 출처가 없으면 컴포넌트 자체를 렌더하지 않습니다.

## 6. 배치 규칙
- 항상 **해당 콘텐츠 바로 아래, Layout 푸터(공식 채널 바로가기) 위**에 둡니다.
- 한 화면에 **출처가 하나면 한 줄로 합칩니다.** 현재 `/club`은 경기·순위표·상대전적·최근 폼의 수집처가 전부 다음 스포츠라 `경기 데이터 출처: 다음 스포츠` 한 줄입니다.
- 앞으로 한 화면에서 **수집처가 갈리면 줄을 나눕니다** — `gap-1`로 쌓고 한 줄에 합치지 않습니다. 데이터마다 출처가 다르다는 사실이 드러나야 하기 때문입니다.
