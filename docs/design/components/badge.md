# 컴포넌트: Badge (배지·태그)

- **문서 버전**: v1.0 · **작성일**: 2026-07-11
- **연관 문서**: DESIGN.md, 기능명세서.md (F-01)
- **한 줄 요약**: '대표곡' 같은 짧은 표시를 작은 둥근 라벨로 보여줍니다.

---

## 1. 개요
- 응원가 카드에 붙는 **작은 표식**입니다. 대표적으로 '대표곡' 표시에 사용합니다.
- 나중에 카테고리(팀 응원가/선수 응원가)를 태그로 보여줄 때도 재사용할 수 있습니다.

## 2. 사용처
- F-01 목록 화면의 응원가 카드 (대표곡 표시)

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| text | 배지에 보일 글자 (예: "대표곡") | O |
| variant | 색상 종류 (기본: primary) | X |

## 4. 종류 (Variants)
| 종류 | 배경 | 글자 | 용도 |
|------|------|------|------|
| primary | `#2E7DFF` (인천 블루) | `#FFFFFF` | 대표곡 등 강조 |
| neutral | `#1E232C` | `#A8B0BD` | 일반 카테고리 태그 |

## 5. 스타일 규칙 (DESIGN.md 기준)
- 모서리: 9999px (둥근 pill) 또는 4px (각진 태그) — 대표곡은 pill 권장
- 글자: 12px / 700 / Pretendard
- 안쪽 여백: 2px(위아래) 8px(좌우)

## 6. 사용 예시 (의사코드)
```
<Badge text="대표곡" variant="primary" />
```

## 7. AI 지시용 스니펫
> "배지: `#2E7DFF` 배경, 화이트 글자, 12px Pretendard 700, 9999px pill, 여백 2px 8px. 텍스트는 '대표곡'."
