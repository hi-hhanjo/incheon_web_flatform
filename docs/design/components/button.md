# 컴포넌트: Button (버튼)

- **문서 버전**: v1.0 · **작성일**: 2026-07-11
- **연관 문서**: DESIGN.md, 기능명세서.md (F-02)
- **한 줄 요약**: 주요 동작을 위한 pill 버튼과, 아이콘만 있는 원형 버튼(뒤로 가기 등) 두 종류를 정의합니다.

---

## 1. 개요
- 이 서비스의 버튼은 두 종류입니다.
  - **Primary(주요 버튼)**: 텍스트가 있는 둥근 pill 버튼. (예: "목록으로 가기")
  - **Icon(아이콘 버튼)**: 아이콘 하나만 있는 원형 버튼. (예: 뒤로 가기 ←)

## 2. 사용처
- Icon: F-02 상세 화면의 뒤로 가기 버튼
- Primary: 에러 화면의 "목록으로" 버튼 등 공용

## 3. 입력 (Props)
| 이름 | 설명 | 필수 |
|------|------|------|
| label | 버튼에 보일 글자 (Primary) | Primary만 |
| icon | 아이콘 (Icon 버튼) | Icon만 |
| onClick | 눌렀을 때 실행할 동작 | O |
| disabled | 비활성 여부 (기본 false) | X |

## 4. 상태 (States)
| 상태 | 처리 |
|------|------|
| 기본 | 아래 스타일 규칙대로 표시 |
| 호버/누름 | 배경색이 `#1F6BEA`로 변함 (Primary) / 살짝 밝아짐 (Icon) |
| 비활성(disabled) | 흐리게(투명도↓) 표시, 클릭 안 됨 |

## 5. 스타일 규칙 (DESIGN.md 기준)
### Primary (주요 버튼)
- 배경: `#2E7DFF` (인천 블루) / 글자: `#FFFFFF`
- 모서리: 9999px (완전 둥근 pill)
- 안쪽 여백: 10px(위아래) 20px(좌우)
- 글자: 15px / 600 / Pretendard
- 호버·누름: 배경 `#1F6BEA`

### Icon (아이콘 버튼)
- 배경: `#1E232C` / 아이콘: `#FFFFFF`
- 모서리: 50% (원형)
- 안쪽 여백: 12px
- **최소 터치 크기 44px** 확보 (F-03)

## 6. 사용 예시 (의사코드)
```
<IconButton icon="←" onClick={목록으로} />
<Button label="목록으로 가기" onClick={목록으로} />
```
